##  Sobre o Projeto

Este é um **Jogo da Cobrinha (Snake)** desenvolvido em **Python**, utilizando a biblioteca **Pygame**. O objetivo deste projeto foi criar uma aplicação interativa e divertida como parte de uma **apresentação escolar** na disciplina de **Tecnologia e Inovação**, durante o **1º ano do Ensino Médio**.

O jogo segue a lógica clássica:
- A cobra se move pelo cenário controlada pelas setas do teclado.
- Ela cresce conforme come as comidas que aparecem aleatoriamente na tela.
- O jogo termina caso a cobra colida consigo mesma ou com as bordas da janela.

---

##  Estrutura do Repositório

- `main.py` — Script principal com a lógica e execução do jogo.
- `sprite.py`, `classes.py`, `texte.py`, `teste.py`, entre outros — módulos auxiliares que estruturam elementos como sprites, lógica e testes.
- Arquivos de áudio (`morte.mp3`, `smw_coin.mp3`, etc.) — efeitos sonoros utilizados durante o jogo.
- `README.md` — documentação do projeto.
